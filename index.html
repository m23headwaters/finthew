Personal Finance Dashboard

A secure personal finance management tool

About This Project

This personal finance dashboard securely connects to financial institutions like American Express and Charles Schwab to provide consolidated account visibility and financial insights.

Features

  • Secure account aggregation via Plaid API
  • Real-time balance and transaction monitoring
  • Personal financial analytics
  • Privacy-focused design for personal use

Data Security

This application uses Plaid's secure API to connect to financial institutions. No financial credentials are stored locally.
